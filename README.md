# Personal Portfolio

A modern, fast, and responsive **personal portfolio website** built with
**React**, **Tailwind CSS**, and **FormSubmit**.\
This portfolio showcases my projects, skills, experience, and provides
an easy way for visitors to contact me.

------------------------------------------------------------------------

##  Tech Stack

-   **React**
-   **Tailwind CSS**
-   **FormSubmit** -- Secure & simple email handling for the contact
    form

------------------------------------------------------------------------

##  Features

-   Fully responsive modern UI\
-   Animated sections and smooth scrolling\
-   Projects showcase with clean layout\
-   Contact form integrated with FormSubmit\
-   Fast loading & optimized performance
